# Phase 1: 理論基礎強化 — 研究報告

## 1. 核心理論框架比較

### 1.1 Steedman (1984) vs. Rohrmeier (2020) 文法比較

| 特徵 | Steedman 1984 | Rohrmeier 2020 |
|------|---------------|----------------|
| **文法類型** | Context-sensitive rewrite rules | Abstract context-free grammar (ACFG) |
| **核心操作** | 6條替代規則 (substitution rules) | 2種依賴關係: prolongation + preparation |
| **處理範圍** | 12-bar blues, rhythm changes | 全面爵士和聲 (standards, AABA forms, blues) |
| **調性處理** | 相對音程步進 | Key features 附加到符號 |
| **tritone substitution** | 作為替代規則 | V(7)/X → V(7)key=♭V/X/Y (保留功能解釋) |
| **階層結構** | 較淺層 | 深層遞歸嵌套 |
| **形式化程度** | 半形式化 | 完整形式化，可計算實作 |

### 1.2 Rohrmeier 框架核心規則

**Prolongation (延長)**
```
X → X X  (coordination)
X → X' X (weak prolongation, different voicing)
```

**Preparation (預備)**
```
X → Prep/X X (general form)
X → V/X X   (applied dominant)
X → Δ/X X   (diatonic fifth: ii-V-I)
V → IV V    (subdominant preparation)
```

**Substitution (替代)**
```
X → Sub/X (unary rule)
V(7)/X → V(7)key=♭V/X/Y (tritone substitution)
V(7)/X → V(7)key=♭III/X/Y (backdoor dominant)
```

### 1.3 Tymoczko 幾何框架整合點

- **Orbifold T^n/S_n**: 和弦空間的非歐幾何表示
- **Voice leading as line segments**: 聲部導進 = 空間中的線段
- **Near-symmetry**: 三全音替代的幾何基礎 — 減七和弦位於 orbifold 中心
- **Jazz 應用**: A Geometry of Music (2011) Chapter 10 專門討論爵士和聲

### 1.4 Cecchetti et al. (2022) 感知實驗關鍵發現

**Octatonic Equivalence Classes (OECs)**
- T (Tonic): {0, 3, 6, 9} — C, E♭, G♭, A
- D (Dominant): {2, 5, 8, 11} — D, F, A♭, B  
- S (Subdominant): {1, 4, 7, 10} — D♭, E, G, B♭

**實驗結果**
- 100 名受試者對爵士和弦進行評分
- 支持 OEC 理論：同一等價類內的和弦替代被感知為功能相似
- 為規則系統提供認知心理學依據

## 3. 理論整合要點

### 3.1 規則系統設計的理論基礎

1. **功能和聲語法** (Rohrmeier)
   - preparation 和 prolongation 作為核心操作
   - 遞歸嵌套產生複雜進行

2. **幾何約束** (Tymoczko)
   - 聲部導進最小化作為優化目標
   - 和弦空間距離影響進行選擇

3. **認知依據** (Cecchetti)
   - OEC 等價類有感知基礎
   - 替代規則不是任意的

### 3.2 待解決的理論問題

1. **Post-bop 非功能和聲**: Rohrmeier 框架需要 "contrast" 關係擴展
2. **Modal interchange 的形式化**: borrowing 規則的限制條件
3. **PCFG 權重學習**: 如何從語料庫學習規則概率
